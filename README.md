# userscripts

A personal collection of [userscripts](https://en.wikipedia.org/wiki/Userscript) I've written over the years. Use with a userscript manager like [Tampermonkey](https://www.tampermonkey.net/), [Violentmonkey](https://violentmonkey.github.io/), or [Userscripts App (iPhone)](https://apps.apple.com/us/app/userscripts/id1463298887).

Each script lives in its own folder with a dedicated README. Open the script's `.user.js` file in the GitHub UI and click the **Raw** button. Your userscript manager should offer to install it.

## Scripts

| Script                                                           | Description                                                                                                      | Sites       |
| ---------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- | ----------- |
| [YouTube Macros](./youtube-macros/)                              | Step playback speed, hide chrome for clean fullscreen, and download the current frame (key + control-bar button) | youtube.com |
| [World Gym Guest Form Autofill](./worldgym-guest-form-autofill/) | Auto-fills the World Gym Quebec guest registration form                                                          | ggpx.info   |
